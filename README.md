## Quick Start

```bash
# Clone the repo
git clone https://github.com/Karan-Singh-Bisht/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start dev server at http://localhost:5173
npm run dev

# Build for production
npm run build && npm run preview
```
